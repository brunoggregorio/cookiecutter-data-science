.. contents::

.. highlight:: console

Documentação da API gerada de forma automática para a linguagem Python
através da extensão `sphinx-autodoc`.

.. note:: 
   Para outras linguagens, a geração automática de documentação 
   pode ser adicionada como uma extensão ao **sphinx**, ou pode ser gerada
   paralelamente e importada para cá, como por exemplo usando o `Doxygen`__.

__ https://www.doxygen.nl/index.html
